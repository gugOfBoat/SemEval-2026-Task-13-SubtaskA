## Overview

This repository contains the complete pipeline for **SemEval-2026 Task 13 – Subtask A**, including data downloading, exploratory data analysis, model training, and evaluation.

The pipeline fine-tunes a pre-trained transformer model (default: `bert-base-uncased`) for the Subtask A classification objective.

## Setup

### Prerequisites
- [Conda](https://docs.conda.io/en/latest/) (Miniconda or Anaconda)
- A [Kaggle](https://www.kaggle.com) account with an API token

### Automated setup

```bash
bash prepare.sh
```

`prepare.sh` will:
1. Create all required project directories
2. Create the `semeval2026` Conda environment from `environment.yml`
3. Prompt for your Kaggle credentials and write them to `.env`

### Manual setup

```bash
conda env create -f environment.yml
conda activate semeval2026
```

Then create a `.env` file (or place `kaggle.json` in `~/.kaggle/`):

```
KAGGLE_USERNAME=your_kaggle_username
KAGGLE_KEY=your_kaggle_api_key
```

> **Important:** Never commit your `.env` or `kaggle.json` to version control.
> Generate your Kaggle API token at <https://www.kaggle.com/settings>.

## Dataset

Download the dataset with:

```bash
conda activate semeval2026

```

Files are saved as `.parquet` in the `data/` directory.

Optional arguments:
```
--dataset   Kaggle dataset slug  (default: gugofboat/semeval-2026-task-13)
--dest      Output directory     (default: data/)
```

## Usage

### 1. Exploratory Data Analysis

```bash
python info_dataset/info_dataset_subTaskA.py
```

Produces plots in `img/img_TaskA/`:
- `class_distribution_<split>.png`
- `text_length_distribution_<split>.png`
- `missing_values_heatmap_<split>.png`

### 2. Subtask A – End-to-end pipeline

```bash
# Full pipeline (preprocess → train → evaluate)
python src/src_TaskA/pipeline.py

# Individual steps
python src/src_TaskA/preprocess.py
python src/src_TaskA/train.py --model_name bert-base-uncased --epochs 3
python src/src_TaskA/evaluate.py
```

Key training options:
| Option | Default | Description |
|---|---|---|
| `--model_name` | `bert-base-uncased` | Hugging Face model identifier |
| `--epochs` | `3` | Number of training epochs |
| `--batch_size` | `16` | Per-device batch size |
| `--lr` | `2e-5` | Learning rate |
| `--output_dir` | `models/TaskA` | Directory for saved model |
